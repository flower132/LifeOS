# 《LifeOS Wireframe V1.0》

> **这份文档是 LifeOS 的低保真线框图。**
> 它不用真实的视觉设计，只用文字和简单图形描述每个页面的结构、布局和信息层级。
> 任何设计师可以基于这份 Wireframe 输出高保真 UI。

---

# 设计说明

## Wireframe 约定

```
┌─────────────────────┐  ← 手机屏幕外框
│                     │
│  [文字内容]          │  ← 界面元素
│                     │
│  ┌─────────────┐    │  ← 卡片/区域
│  │  卡片内容   │    │
│  └─────────────┘    │
│                     │
│  ─────────────────  │  ← 分隔线
│                     │
│  [按钮] [按钮]      │  ← 操作按钮
│                     │
└─────────────────────┘
```

## 字体大小约定（Wireframe 中）

- `大标题`：页面主标题
- **粗体**：重要信息
- 普通文字：正文
- `小字`：辅助信息、时间、标签
- `[按钮]`：可点击元素

# 结语

这份 Wireframe 用低保真方式定义了 LifeOS 所有核心页面的结构、布局和信息层级。

它没有讨论颜色、字体、图标、动画细节，只关注：
- 页面上有什么
- 它们如何组织
- 用户先看什么、后看什么
- 每个元素为什么存在

设计师可以基于这份 Wireframe 输出高保真 UI，开发者可以基于它实现页面结构。
